# 📊 E-Commerce Sales Analytics Dashboard

An end-to-end data analytics project that transforms raw e-commerce transaction data into actionable business insights using SQL and Power BI.

This project demonstrates the complete analytics workflow—from data preparation and relational database design to business KPI analysis and interactive dashboard creation. The objective is to simulate the work of a Data Analyst by answering real-world business questions related to sales, customers, products, and business performance.

---

## 🎯 Objectives

* Analyze large-scale e-commerce sales data using SQL.
* Design a relational database with multiple interconnected tables.
* Generate business KPIs and actionable insights.
* Build an interactive Power BI dashboard for decision-making.
* Document the complete analytics workflow and findings.

---

## 🛠️ Tech Stack

* **Database:** MySQL
* **Query Language:** SQL
* **Data Processing:** Python (Pandas)
* **Visualization:** Power BI
* **Version Control:** Git & GitHub

---

## 📂 Dataset

**Dataset:** Olist Brazilian E-Commerce Public Dataset (Kaggle)

The dataset contains multiple related tables representing an e-commerce platform, including:

* Customers
* Orders
* Order Items
* Products
* Sellers
* Payments
* Reviews
* Categories
* Geolocation
